# Ghana SHS Grading Sheet & WAEC STP Requirements

## WAEC Grading Scale (WASSCE)

| Grade | Numeric Value | Score Range (%) | Interpretation |
|-------|---------------|-----------------|----------------|
| A1    | 1             | 75 – 100        | Excellent      |
| B2    | 2             | 70 – 74         | Very Good      |
| B3    | 3             | 65 – 69         | Good           |
| C4    | 4             | 60 – 64         | Credit         |
| C5    | 5             | 55 – 59         | Credit         |
| C6    | 6             | 50 – 54         | Credit         |
| D7    | 7             | 45 – 49         | Pass           |
| E8    | 8             | 40 – 44         | Pass           |
| F9    | 9             | 0 – 39          | Fail           |

**Assessment Breakdown (all subjects):**
- Continuous Assessment (School-Based — SBA): 30%
- External Examination (WAEC): 70%
- Final Score = SBA + Exam (max 100%)

---

## Universal Assessment Architecture (SHS 1–3)

Every SHS subject — Core, Elective, Technical, or Vocational — records two data fields:

| Component | Weight | What is recorded |
|-----------|--------|-----------------|
| SBA (School-Based Assessment) | 30% | Class Tests · Projects/Fieldwork · Homework · End-of-Term Tests |
| WAEC External Examination | 70% | Paper 1 (Objective) · Paper 2 (Written Theory) · Paper 3 (Practical/Oral, if applicable) |

Both components are captured and forwarded to WAEC STP for final transcript generation.

### Per-Student Row (Standard Formatting)

| Field | Data type | Description |
|-------|-----------|-------------|
| `ID` | string | Student ID / Index Number |
| `Name` | string | Full name |
| `SBA (30%)` | number 0–30 | Teacher-recorded composite score from Term 1–3 SBA tasks |
| `Exam - Section A` | number | Paper 2 obj/structured score (0–40 typical) |
| `Exam - Section B` | number | Paper 2 essay/applied score (0–60 typical) |
| `Exam - Section C` | number (opt.) | Extra section for English/Visual/3-D arts (0–40 typical) |
| `Exam Total Raw` | number | Sum of all exam sections |
| `Exam % → scaled` | number | `(Exam Total / maxRaw) × 70` |
| `Final Score` | number | `SBA + Exam %` (0–100) |
| `Grade` | string A1–F9 | Numeric threshold lookup |
| `Remark / Smart Remark` | string | GES comment (auto-generated per grade) |

### Score / Grade Calculation

```
ExamPct   = (ExamTotalRaw / SubjectMaxRaw) × 70
Final     = SBAScore + ExamPct           →  0 ≤ Final ≤ 100
Grade:
  ≥75 → A1    ≥70 → B2   ≥65 → B3   ≥60 → C4   ≥55 → C5
  ≥50 → C6    ≥45 → D7   ≥40 → E8   <40 → F9
```

## University Admission Requirements (Ghana)

### Minimum Requirements
- **Five (5) Credit Passes** (A1–C6) including:
  - English Language
  - Core Mathematics
  - Integrated Science (for Science programmes) OR Social Studies (for Arts/Business programmes)
  - Three (3) relevant elective subjects

## WAEC STP (Student Transcript Portal) — Per-Subject Data Entry

| STP Field | What it captures (per subject per student) |
|-----------|---------------------------------------------|
| Learner Bio-data | Student name · Index No. · Date of birth · Gender |
| Subject Combination | All subjects registered for that student (min 7 for WASSCE eligibility) |
| Learning Area | e.g., "GENERAL ARTS", "BUSINESS", "HOME ECONOMICS" |
| Year / Form | SHS 1, SHS 2, or SHS 3 |
| Semester | 1st / 2nd / 3rd |
| SBA Score | Numeric score 0–30 (30% max) — accepted by system as-is |
| WAEC Exam Score | Paper 1 + Paper 2 (+ Paper 3 if applicable) — typically 70 max |
| Final Score | SBA + Exam (0–100) — WAEC validates cannot exceed 100 |
| Grade | A1 … F9 |
| GPA per Term | (Sum of grade-points for that term's subjects ÷ count) |
| CGPA | Cumulative across all SHS terms |
| Qualitative Assessment | Character qualities · Leadership · Discipline · Teamwork · Ethics — rated 1–5 |

## STP Data Entry Checklist

- [ ] Student bio-data captured  
- [ ] All subjects registered (minimum 7 for WASSCE eligibility)  
- [ ] For each subject: SBA score (School) and Exam score (WAEC) entered  
- [ ] Scores validated (SBA ≤ 30%, Exam ≤ 70%, Final ≤ 100%)  
- [ ] Final grade mapped using WAEC A1–F9 table  
- [ ] Practical/Oral scores recorded for Practical subjects (Bio, Chem, Physics, Agricultural Science, Food & Nutr, Clothing & Text, Visual Arts, Auto Mechanics, Woodwork, etc.)  
- [ ] GPA per Term computed  
- [ ] CGPA updated  
- [ ] Qualitative/behavioural observations rated 1–5  
- [ ] HOD review & certification submitted  
- [ ] Transcript generated for review
