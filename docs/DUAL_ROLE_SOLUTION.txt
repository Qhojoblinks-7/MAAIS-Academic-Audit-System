## MAAIS Dual-Role HOD/Teacher Solution
### Date: 2026-06-22

### PROBLEM SUMMARY

When a teacher is assigned as HOD, the current system:
1. Changes their role from TEACHER → HOD (single role enum)
2. Removes their teaching assignments from view
3. Prevents them from accessing their own teaching dashboard
4. Creates redundant frontend/backend systems for HOD vs Teacher

### RECOMMENDED SOLUTION: Context-Based Interface

**Maintain single role but add dual-context capability via database flags**

### DATABASE CHANGES

Add to `StaffProfile` model in `schema.prisma`:
```
hodDepartmentId       String?   // Department they oversee (can be same or different)
canTeach              Boolean   @default(true) // Can they actively teach?
isHod                 Boolean   @default(false) // Quick check for HOD status
```

### BACKEND CHANGES

#### 1. Users Service Update
```typescript
// src/users/users.service.ts
async getAllStaff(user?: { id: string; role: Role }) {
  let departmentId: string | undefined;

  if (user?.role === Role.HOD || user?.isHod) {
    const staff = await this.prisma.staffProfile.findUnique({
      where: { userId: user.id },
    });
    departmentId = staff?.hodDepartmentId || staff?.departmentId;
  }

  return this.prisma.staffProfile.findMany({
    where: {
      ...(departmentId ? { departmentId } : {}),
    },
    include: {
      user: { select: { email: true, role: true, isActive: true } },
      department: true,
      teachingAssignments: { include: { subject: true, classSection: true } },
    },
    orderBy: { lastName: 'asc' },
  });
}
```

#### 2. New Context Endpoint
```typescript
// src/users/users.controller.ts
@Get('me/context')
@Roles(Role.TEACHER, Role.HOD)
async getMyContext(@CurrentUser('id') userId: string, @CurrentUser('role') role: Role) {
  const profile = await this.prisma.staffProfile.findUnique({
    where: { userId },
    include: { 
      department: true,
      hodDepartment: true, // New relation
      teachingAssignments: { 
        include: { subject: true, classSection: true } 
      }
    }
  });

  return {
    role,
    department: profile?.department,
    hodDepartment: profile?.hodDepartment,
    teachingAssignments: profile?.teachingAssignments,
    canTeach: profile?.canTeach ?? true,
    canOversight: role === Role.HOD || profile?.hodDepartmentId != null,
    activeMode: role === Role.HOD && profile?.canTeach ? 'dual' : 'single'
  };
}
```

const refreshDepartmentProgress = useCallback(async () => {
  if (!isHod) return;
  
  if (viewMode === 'teaching') {
    // Show only their teaching assignments
    const assignments = await hodService.getMyTeachingAssignments();
    setDepartmentProgress(assignments);
  } else {
    // Show full department oversight
    const data = await hodService.getDepartmentProgress();
    setDepartmentProgress(data.items || []);
  }
}, [isHod, viewMode]);
