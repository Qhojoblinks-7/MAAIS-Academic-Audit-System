Professional PWA icons for MAAIS Academic Audit System

Live Preview

How to Use

  1. Click any download button to save the PNG icon
  2. Rename the file to match the manifest entry (e.g., pwa-192x192.png)
  3. Place icons in front-end/public/
  4. Update vite.config.js manifest paths if needed
  5. Build and test: npm run build && npm run preview
